Deepin零散记录

2020-09-05 Deepin Deepin 0 Comments Word Count: 276(words) Read Count: 1(minutes)

有新发现就更新。

  1. 本人是deepin v20。deepin的tty1默认是图形界面，而一般Xorg默认在tty7。原因解释见论坛：请问15.5是默认tty1是图形界面么？

    • 官方回答: 目前默认是tty1，　原因是为了同plymouth(启动画面)使用的tty1，进行无缝自然过渡。也就是Xorg和plymouth直接使用同一个tty，避免创建新的tty造成进入X11那一瞬间的黑屏现象。

  2. Deepin深度影院无法打开的问题
    来自论坛解答 深度影院无法打开，关闭深度影院的硬件加速即可。
  3. IDEA最近总是莫名其妙卡死的原因：搜狗输入法最新版——“搜狗输入法个人版v2.3.2”需要背锅，详细见论坛贴：[搜狗输入法个人版]goland和IDEA直接卡死，无法正常使用
  4. deepin星火商店有个AppImage转Deb的转换器，转换到*.deb，就能让软件安装到系统，不用再便携式运行了。另外论坛发现帖子：请问AppImage .snap .deb Flatpak 区别
